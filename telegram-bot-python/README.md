# Python Telegram Bot - Быстрый старт

## 🚀 Установка и запуск

### 1. Установите Python 3.9+

### 2. Установите зависимости:
```bash
cd telegram-bot-python
pip install -r requirements.txt
```

### 3. Запустите бота:
```bash
python bot.py
```

## 📦 Деплой на Railway.app (Рекомендуется)

1. Зарегистрируйтесь на https://railway.app
2. Создайте новый проект
3. Подключите GitHub репозиторий
4. Укажите папку: `telegram-bot-python`
5. Добавьте переменные окружения из `.env`
6. Railway автоматически задеплоит бота

## 🔧 Переменные окружения

Все переменные уже в `.env` файле:
- TELEGRAM_BOT_TOKEN
- CRYPTOBOT_API_TOKEN
- FIREBASE_ADMIN_PROJECT_ID
- FIREBASE_ADMIN_CLIENT_EMAIL
- FIREBASE_ADMIN_PRIVATE_KEY

## ✨ Особенности Python бота

- ⚡ Быстрый отклик (без таймаутов)
- 🎨 Премиум эмодзи в сообщениях и кнопках
- 💰 Интеграция с CryptoBot
- 🔥 Интеграция с Firebase
- 🌐 Поддержка русского языка

## 📝 Команды бота

- `/start` - Главное меню
- `/balance` - Проверить баланс
- `/buy` - Купить кредиты
- `/help` - Помощь
